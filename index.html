Artefak #1: RPP Modul VLSM

Advanced Network Segmenting: Variable Length Subnet Mask

Target: Fase F

Konteks & Tujuan

Topik VLSM merupakan fondasi kritis dalam administrasi jaringan enterprise. Pembelajaran ini dirancang untuk mengembangkan kemampuan siswa dalam mengalokasikan alamat IP secara efisien sesuai kebutuhan subnet heterogen. Tujuan utama adalah mengubah pemahaman siswa dari level hapalan menjadi level analisis dan aplikasi kontekstual.

Teori Pedagogis

Menggunakan pendekatan Understanding by Design (UbD) dan Zone of Proximal Development (ZPD) Vygotsky. Pembelajaran dimulai dari konsep biner sederhana, kemudian diperancah menuju pemahaman matematis kompleks alokasi subnet optimal.

Kelebihan Pembelajaran
  • ✓ Scaffolding terstruktur dari konsep dasar hingga aplikasi kompleks
  • ✓ Integrasi visualisasi diagram dan simulasi Packet Tracer
  • ✓ Aktivitas hands-on dengan perangkat router fisik Mikrotik
  • ✓ Penilaian formatif berkelanjutan untuk feedback real-time
Tantangan & Perbaikan
  • • Beberapa siswa masih kesulitan konversi biner cepat
  • • Waktu alokasi lab terbatas untuk eksperimen mendalam
  • • Diperlukan penambahan media drill interaktif online
  • • Perlu kolaborasi dengan industri untuk studi kasus real
Artefak #2: Media Pembelajaran OSPF

Dynamic Gateway Rerouting: OSPF & Algoritma Dijkstra

Video & Simulasi

Format Media

Kombinasi video demonstrasi (15 menit) dengan simulasi interaktif di Packet Tracer dan Mikrotik RouterOS. Video menampilkan langkah demi langkah konfigurasi dengan penjelasan konsep matematis cost metric secara paralel. Simulasi memungkinkan siswa melakukan eksperimen perubahan topologi secara real-time.

Outcome Pembelajaran

Setelah mempelajari media ini, siswa mampu: (1) Memahami algoritma Dijkstra dan aplikasinya dalam OSPF, (2) Menghitung cost metric berdasarkan bandwidth interface, (3) Mengkonfigurasi OSPF di lab fisik dengan modifikasi manual cost, (4) Troubleshooting masalah konvergensi routing dinamis.

Inovasi Media
  • ✓ Animasi matematis cost calculation yang interaktif
  • ✓ Integrasi suara dalam bahasa lokal untuk aksesibilitas
  • ✓ Subtitle dan transcript untuk review mandiri
  • ✓ Quiz embedded untuk self-assessment
Peningkatan Berkelanjutan
  • • Update konten sesuai rilisan terbaru RouterOS
  • • Tambahan skenario failover otomatis (BFD)
  • • Integrasi dengan lab fisik siswa untuk capstone
  • • Repository GitHub untuk versi komunitas terbuka
Artefak #3: Praktik Lab Fiber Optics

Penyambungan Serat Optik & Kalibrasi Fusion Splicer

Praktik Laboratorium

Desain Praktik

Praktik ini dirancang sebagai pembelajaran terintegrasi yang menghubungkan fisika optik (Hukum Snellius, Total Internal Reflection) dengan teknik praktis splicing. Siswa tidak hanya melakukan penyambungan, tetapi juga memahami fenomena fisik di balik proses tersebut melalui analisis nilai pelayangan sinyal.

Konstruksi Kognitif

Menggunakan pendekatan Project-Based Learning (PjBL) yang terikat pada prinsip hibridasi industri-akademik. Siswa menganalisis data pelemahan sinyal real dari jaringan SMKN 1, kemudian mengaplikasikan pengetahuan untuk optimasi maintenance preventif jaringan sekolah.

Aspek Keberlanjutan
  • ✓ Penggunaan kembali serat optik bekas untuk latihan
  • ✓ Dokumentasi praktik mencegah pemborosan material
  • ✓ Integrasi dalam Smart Lab untuk IoT monitoring
  • ✓ Penghijauan efektivitas energi melalui optik
Metrik Pencapaian
  • • 95% siswa berhasil splicing dengan loss < 0.3 dB
  • • Rata-rata waktu splicing turun dari 12 menit menjadi 4 menit
  • • 88% siswa mampu menjelaskan Hukum Snellius secara verbal
  • • Portfolio foto praktik dokumentasi 100% lengkap
