Skenario Ekstrem: Injeksi Drop-Paket & Kelumpuhan Lab

Pada pelaksanaan Modul Ajar 7, saya sengaja menyuntikkan skenario gangguan ekstrem tanpa pemberitahuan kepada siswa. Di tengah pengerjaan proyek, saya memicu skrip otomasi jarak jauh yang mensimulasikan kegagalan tautan utama ISP (Dual-ISP Link Failure) disertai pembanjiran tabel memori perangkat (MAC-Flooding Storm).

# CRITICAL EVENT LOG TRIGGERED [FATAL] Gateway ISP-1 (192.168.10.1) -> STATUS: RTO (REQUEST TIMEOUT) [WARNING] Core Switch CAM Table Load: 99.8% -> MEMORY OVERFLOW IMMINENT [ACTION] Activating Student Incident Response Protocol...

Seketika, seluruh komputer klien kehilangan akses internet. Grafik telemetri jatuh ke angka nol. Di sinilah letak ujian sesungguhnya: mengubah kepanikan ruang kelas menjadi medan tempur siber terstruktur, di mana ketahanan psikomotorik dan nalar logika analitis siswa diuji hingga batas maksimal.
