Dynamic Gateway Rerouting: OSPF & Algoritma Dijkstra

Target Capaian: Fase F
# Mathematical Engineering Formulation:

// Rumus Perhitungan Cost Metric OSPF (Algoritma Shortest Path First - Dijkstra):

Cost = Reference Bandwidth (10^8 bps) / Interface Bandwidth (bps)

// Perbandingan Cost Jalur Fast Ethernet (100 Mbps) vs Gigabit Ethernet (1 Gbps):

Cost_FastEthernet = 100,000,000 / 100,000,000 = 1

Cost_GigabitEthernet = 100,000,000 / 1,000,000,000 = 0.1 → Dibulatkan menjadi Cost Mutlak = 1

Modifikasi Manual Cost Diperlukan agar Penentuan Jalur Utama Router Tidak Mengalami Kegagalan Ambiguitas Konvergensi.

Bug Kognitif Tradisional (Desain Awal): Konfigurasi OSPF diajarkan hanya sebatas mengetikkan perintah `network` dan `area` pada simulator siber Packet Tracer tanpa memaparkan bagaimana bobot matematis cost mempengaruhi path selection dan convergence time.

The Quad-Pillar Manifesto: Melampaui Batas Tradisional Vokasi

P1 Tekno-Pedagogis Kontemporer (Konvergensi Industri-Socrates)

Teknologi di dalam laboratorium SMK tidak boleh lagi diposisikan sekadar sebagai media instruksional pasif atau alat simulasi tanpa nyawa. Teknologi adalah Epistemic Tool yang mengubah cara peserta didik berpikir, berargumen, dan memvalidasi pengetahuan dalam konteks engineering riil. Setiap perangkat keras (router Mikrotik, Fusion Splicer, Optical Power Meter) adalah narasumber pembelajaran yang bercerita tentang prinsip-prinsip fundamental fisika, matematika, dan informasi.

Jika tujuan esensial adalah pemahaman relasional tentang bagaimana jaringan modern beradaptasi terhadap gangguan, maka teknologi yang dipilih adalah yang memungkinkan eksperimen empiris autentik, bukan simulasi kering.
