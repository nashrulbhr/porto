1. Manifesto Kompetensi dan Evolusi Pedagogis

"Transformasi diri saya dari seorang R&D Staff sistem udara otonom UAV di industri aviasi geospasial menjadi seorang pendidik klinis di SMK Negeri 1 Surabaya bukanlah sebuah perpindahan karir linier yang sederhana. Ini adalah sebuah proses re-arsitektur kognitif total."

Modul Ajar 1 sampai dengan 6 adalah prasasti digital dari fase transisi awal saya. Perangkat-perangkat tersebut merekam bagaimana saya pada mulanya masih berpikir seperti seorang engineer murni yang kaku—menyusun materi infrastruktur jaringan sebagai tumpukan instruksi teknikal yang padat, dingin, dan mengabaikan keragaman kesiapan belajar (*readiness*) siswa.

Pada mulanya, saya berasumsi bahwa mentransfer keahlian *Variable Length Subnet Mask (VLSM)* atau konfigurasi *OSPF Dynamic Routing* cukup dengan memberikan demonstrasi sintaks CLI (*Command Line Interface*) satu arah di papan tulis. Akibatnya, terjadi diskoneksi kognitif di dalam laboratorium: siswa kelompok bawah mengalami *analysis paralysis* (kelumpuhan analisis biner), sementara siswa kelompok atas bosan karena kehilangan tantangan kontekstual yang nyata.

Kesadaran klinis ini mengarahkan saya pada rekonstruksi kurikulum total menggunakan pendekatan **Understanding by Design (UbD)** melalui strategi *Backward Design*. Saya membalikkan paradigma berpikir: menetapkan hasil akhir yang esensial (kemampuan siswa mengamankan jaringan dari serangan krisis nyata), merancang instrumen penilaian autentik, baru kemudian menurunkan aktivitas pembelajaran berdiferensiasi yang diakomodasi oleh perancah kognitif (*Scaffolding*) berbasis teori *Zone of Proximal Development (ZPD)* dari Vygotsky.

Dynamic Gateway Rerouting: OSPF & Algoritma Dijkstra

Target Capaian: Fase F
# Mathematical Engineering Formulation:

// Rumus Perhitungan Cost Metric OSPF (Algoritma Shortest Path First - Dijkstra):

Cost = Reference Bandwidth (10^8 bps) / Interface Bandwidth (bps)

// Perbandingan Cost Jalur Fast Ethernet (100 Mbps) vs Gigabit Ethernet (1 Gbps):

Cost_FastEthernet = 100,000,000 / 100,000,000 = 1

Cost_GigabitEthernet = 100,000,000 / 1,000,000,000 = 0.1 -> Dibulatkan menjadi Cost Mutlak = 1

Modifikasi Manual Cost Diperlukan agar Penentuan Jalur Utama Router Tidak Mengalami Kegagalan Ambiguitas Konvergensi.

Bug Kognitif Tradisional (Desain Awal): Konfigurasi OSPF diajarkan hanya sebatas mengetikkan perintah `network` dan `area` pada simulator siber Packet Tracer tanpa memaparkan bagaimana bobot matematis *cost* mempengaruhi keputusan tabel routing di dalam memori internal CPU router.
Patch Pedagogis Bermakna (Intervensi Sukses): Penerapan *Context-Based Learning (CBL)*. Saya membawa perangkat keras router fisik Mikrotik modular ke dalam kelas. Siswa sengaja diperintahkan memutuskan kabel jalur serat optik utama laboratorium untuk menyaksikan bagaimana algoritma Dijkstra bekerja secara otonom menghitung ulang jalur alternatif tercepat berdasarkan visualisasi grafik telemetri *Winbox*.

Fisika Optik & Teorema Kapasitas Saluran Shannon-Hartley

Target Capaian: Fase F
# Mathematical Engineering Formulation:

// 1. Hukum Snellius untuk Pemantulan Sempurna (Total Internal Reflection Core-Cladding Serat Optik):

Sudut Kritis (θ_c) = arcsin(n_cladding / n_core) | Jika n_core = 1.48 dan n_cladding = 1.46

θ_c = arcsin(1.46 / 1.48) = arcsin(0.9864) = 80.5 derajat

// 2. Teorema Kapasitas Saluran Shannon-Hartley untuk Batas Atas Bandwidth Kebisingan:

Teknologi di dalam laboratorium SMK tidak boleh lagi diposisikan sekadar sebagai media instruksional pasif atau alat simulasi tanpa nyawa. Teknologi adalah Epistemic Tool—sebuah perpanjangan tangan dari struktur kognitif manusia untuk memanipulasi, mengeksplorasi, dan menguji batas-batas realitas logika siber.
